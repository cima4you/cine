# R-CINE — موقع مشاهدة الأفلام والمسلسلات

موقع عربي لمشاهدة الأفلام والمسلسلات أونلاين. مبني على HTML/CSS/JS ثابت (Static Site) ومنشور على Netlify.

**الرابط**: [https://cima4you.netlify.app](https://cima4you.netlify.app)

---

## قائمة الميزات (16 ميزة)

### 1. 🎨 إعادة تصميم UI كامل (tuktukhd)
تصميم غامق أنيق (#0f0f1a) بهيدر ثابت، قوائم منسدلة، بطاقات متناسقة، وتأثيرات hover.

### 2. 🌐 Netlify
نشر تلقائي من GitHub إلى Netlify مع HTTPS. أي `git push` → نشر فوري.

### 3. 📱 PWA + Service Worker
الموقع قابل للتثبيت كتطبيق على شاشة الجهاز، مع Service Worker للتخزين المؤقت والتشغيل دون إنترنت جزئيًا.

### 4. 🦴 Skeleton Loading
8 بطاقات هيكل عظمي متحركة (shimmer) تظهر أثناء تحميل المحتوى.

### 5. 🖼️ Banner Carousel
شريط دعائي دوار يعرض 6 أفلام مع تنقل تلقائي كل 5 ثواني، وزر مشاهدة يفتح المودال.

### 6. ⭐ المفضلة (Favorites)
زر نجمة ⭐ على البطاقات يحفظ الفيلم في localStorage. تصفية حسب "⭐ المفضلة" لعرضها.

### 7. 🔖 للمشاهدة (Watchlist)
زر 🔖/📑 يحفظ الأفلام/المسلسلات للمشاهدة لاحقًا. تصفية حسب "🔖 للمشاهدة".

### 8. ▶️ مشاهدة مستمرة (Continue Watching)
يحفظ آخر حلقة شاهدتها تلقائيًا في localStorage ويعرضها في قسم مخصص بأعلى الصفحة (آخر 20).

### 9. 📤 أزرار المشاركة
في المودال: واتساب، فيسبوك، إكس (تويتر)، ومشاركة Native للمتصفح.

### 10. 🔍 Search Autocomplete
اقتراحات لحظية (أعلى 10) أثناء الكتابة في مربع البحث.

### 11. 🏷️ تصنيف فرعي (Genre Filter)
بعد اختيار تصنيف رئيسي، تظهر أزرار تصنيف فرعي (أكشن، دراما، كوميدي...) لتصفية المحتوى.

### 12. ⭐ تقييم المستخدمين
تقييم 1-5 نجوم في المودال، محفوظ في localStorage.

### 13. 🔔 إشعارات المتصفح
زر 🔕/🔔 في الهيدر يطلب صلاحية الإشعارات ويرسل إشعارات عبر المتصفح.

### 14. 🗺️ Sitemap + Robots.txt
`sitemap.xml` و `robots.txt` لمحركات البحث.

### 15. ⚙️ Admin Panel
صفحة `admin.html` تعرض إحصائيات المحتوى وبيانات localStorage.

### 16. 👁️ عداد الزوار
عداد في الفوتر يستخدم CountAPI لحساب الزيارات الكلية.

## هيكل الملفات

```
/
├── index.html          # الصفحة الرئيسية (كل الميزات)
├── admin.html          # لوحة التحكم (إحصائيات)
├── manifest.json       # PWA manifest
├── sw.js               # Service Worker
├── sitemap.xml         # خريطة الموقع لمحركات البحث
├── robots.txt          # إعدادات الزحف
├── logo.png            # شعار الموقع
├── data-loader.js      # تحميل بيانات المحتوى (JSON)
├── netlify.toml        # إعدادات نشر Netlify
└── bump-version.ps1    # سكربت رفع الإصدار + git push
```

---

## التخزين المحلي (localStorage)

| المفتاح | الاستخدام |
|---------|-----------|
| `rcine_fav` | عناوين المفضلة (Array) |
| `rcine_wl` | عناوين للمشاهدة (Array) |
| `rcine_cw` | سجل مشاهدة مستمرة (Array) |
| `rcine_ratings` | تقييمات المستخدم (Object) |

---

## النشر

```bash
# تعديل → رفع إصدار → نشر تلقائي
.\bump-version.ps1 "رسالة التحديث"
```

Netlify يراقب فرع `main` وينشر تلقائيًا.
