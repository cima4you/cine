## 3. الخيارات المشتركة (لكلا السكربتين)

| الخيار | المعنى |
|--------|--------|
| `-s`, `--start` | صفحة البداية |
| `-e`, `--end` | صفحة النهاية |
| `-p`, `--page` | رقم صفحة محددة |
| `-sm`, `--start-movie` | رقم فيلم البداية في الصفحة |
| `-em`, `--end-movie` | رقم فيلم النهاية في الصفحة |
| `-m`, `--movie` | رقم فيلم واحد |
| `--all` | كل الأفلام في نطاق الصفحات |
| `--list` | عرض القائمة فقط بدون سحب تفاصيل |
| `--insert` | إدراج مباشر في data.js |
| `-o`, `--output` | ملف الإخراج (يحتفظ بالقديم) |
| `-d`, `--delay` | تأخير بين الطلبات (ثواني) |
| `-u`, `--url` | رابط تصنيف مخصص |

**ملاحظة:** عند استخدام `--all` مع `-o`، إذا كان الملف موجوداً مسبقاً يتم دمج النتائج الجديدة مع القديمة (تجنب التكرار).

### فك تشفير سيرفر المشاهدة

الموقع يستخدم `data-link` المشفر على عناصر `<li class="server--item">`.
السكربت يفك التشفير تلقائياً عبر `decode_server_token()`:
1. **قص** النص عند `0REL0Y&` ← أخذ الجزء الأول
2. **عكس** المحارف
3. **Base64 decode**

النتيجة: رابط مباشر بصيغة `https://megatuktuk.store/iframe/TOKEN`

### أولوية البوستر

السكربت يأخذ البوستر من **صفحة القائمة** (الأكثر دقة) بدل صفحة التفاصيل.

## 4. سكربت المسلسلات الأجنبية (scrape_foreign_series.py)

**الموقع:** `screpte/tuktukhd/scrape_foreign_series.py`
**المصدر:** tuktukhd.com ← مسلسلات أجنبية
**رابط التصنيف:** `https://tuktukhd.com/sercat/مسلسلات-اجنبي/`
**ملف الدمج:** `data/data-foreign-series.js`

### المراحل:
| المرحلة | الوصف |
|---------|-------|
| `listing` | سحب قائمة المسلسلات من صفحات التصنيف |
| `detail` | تفاصيل كل مسلسل (بوستر، وصف، تقييم...) |
| `seasons` | حلقات المواسم |
| `servers` | سيرفرات المشاهدة لكل حلقة |
| `all` | الكل مرة واحدة |

### طرق التشغيل:

```powershell
# سحب الكل (صفحات 1-3) - 4 خيوط متزامنة
python "screpte/tuktukhd/scrape_foreign_series.py" --start 1 --end 3 --workers 4

# استئناف من آخر حفظ (إذا توقف)
python "screpte/tuktukhd/scrape_foreign_series.py" --resume --workers 4

# تحويل ودمج مع ملف البيانات (يضيف الجديد ويحدث الموجود)
python "screpte/tuktukhd/scrape_foreign_series.py" --convert --merge "data/data-foreign-series.js" --output "data/data-foreign-series.js"
```

### ملاحظات السيرفرات:
- مهلة 25 ثانية لكل حلقة (المتعطلة تُتخطى وتُسجل فارغة)
- حفظ تلقائي كل 5 مسلسلات (للاستئناف بـ `--resume`)
- عداد تقدم: `📡 [رقم/الإجمالي] مسلسل - حلقة`

### الدمج:
`merge_with_existing` يدعم الآن **التحديث** (وليس الإضافة فقط):
- التقييم، الجودة، القصة، البوستر، التمثيل، التصنيفات تُحدّث
- المواسم الجديدة تُضاف
- الحلقات الجديدة تُضاف للمواسم الموجودة
- سيرفرات الحلقات الموجودة تُحدّث

## 9. ملاحظات مهمة

- **المتطلبات:** Python 3 + `requests` + `beautifulsoup4`
- **التثبيت:** `pip install requests beautifulsoup4`
- **الترميز:** كل السكربتات تدعم العربية (UTF-8)
- **عدد الصفحات:** الأفلام الأجنبية ~249 صفحة، الأفلام المدبلجة ~5 صفحات
- **كل صفحة:** ~30 فيلم
- **مؤشر التقدم:** السكربت يظهر `[رقم/الإجمالي]` مع تفاصيل كل خطوة
- **الاحتفاظ بالقديم:** عند إعادة التشغيل على نفس ملف الخرج، يدمج النتائج الجديدة مع القديمة تلقائياً
- **تحديث القيم:** سكربتات الدمج (`merge_*.py`) تحدّث السيرفرات والصورة والتقييم... للموجودين، لا تكتفي بالتخطي
- **فك تشفير السيرفرات:** `data-link` في HTML ليس رابطاً مباشراً؛ السكربت يفكه تلقائياً (reverse + base64)
- **أولوية البوستر:** البوستر من صفحة القائمة له أولوية أعلى من صفحة التفاصيل (لتجنب الصور الخطأ)
