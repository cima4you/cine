## سير العمل (Asian Series)

### 1. سحب البيانات
```powershell
python "screpte/ahwak/07062026/scrape_yam_all.py" --cat asian-series --output "screpte/ahwak/ASSIAN/asian_series_full.json" --start-page 1 --end-page 2 --episodes-only
```

### 2. دمج في ملفات الموقع
```powershell
python "screpte/ahwak/ASSIAN/merge_asian_to_data.py"
```
المزايا:
- تنظيف العناوين (إزالة `مسلسل` prefix, `مترجم` suffix)
- إزالة التكرارات (الاحتفاظ بالأكثر حلقات)
- إضافة `isComplete` (مكتمل/مستمر)
- توزيع على ملفين: `completed` (مكتمل) / `ongoing` (مستمر)
- دمج مع تحديث القيم (تحديث السيرفرات والحلقات بدلاً من التخطي)

### 3. التحديث الآلي
```powershell
.\update_ahwak.ps1
```

## سير العمل (Turkish Series)

### 1. سحب البيانات (سكربت شامل — مثل tuktukhd)
```powershell
# الكل: قائمة ← تفاصيل ← سيرفرات ← تحويل ← دمج
python "screpte/ahwak/0606206/scrape_turkish_series.py" --start 1 --end 3 --workers 4 --phase all

# كل مرحلة لحالها
python "screpte/ahwak/0606206/scrape_turkish_series.py" --phase listing    # القائمة فقط
python "screpte/ahwak/0606206/scrape_turkish_series.py" --phase detail     # التفاصيل
python "screpte/ahwak/0606206/scrape_turkish_series.py" --phase servers    # السيرفرات

# استئناف من آخر حفظ
python "screpte/ahwak/0606206/scrape_turkish_series.py" --phase servers --resume --workers 4
python "screpte/ahwak/0606206/scrape_turkish_series.py" --resume

# تحويل ودمج فقط (بدون سحب)
python "screpte/ahwak/0606206/scrape_turkish_series.py" --convert --merge "data/data-turkish-completed.js" --output "data/data-turkish-completed.js"
```

### 2. دمج البيانات الموجودة (سريع — من ملف JSON جاهز)
```powershell
python "screpte/ahwak/0606206/merge_turkish_to_data.py"
```

## إصلاح السيرفرات (2026-06-19)

كانت مشكلة السيرفرات أن جميع الحلقات تقريباً تحتوي على `{name: "watch", url: "watch.php?vid=xxx"}` وهي روابط مشاهدة كاملة لا تصلح كـ embed. السيرفرات الصحيحة موجودة في `turkish_series_full.json` (Vidspeeds, 1vid, Ok, Uqload...).

### سبب المشكلة
- **بيانات قديمة:** `ahwaktv_data_formatted.json` (351 مسلسل) يحتوي سيرفرات وهمية من `watch.php`
- **تداخل جزئي:** ملفات JS (518 مسلسل) تتداخل جزئياً مع full JSON (301 مسلسل) — 194 مشترك فقط
- **تغيير هيكل الموقع:** `see.php` لم يعد يعرض قائمة سيرفرات `<li>` بل يضع iframe مباشر

### الحل
1. **`_fix_servers_v2.py`:** يطابق عناوين المسلسلات (بحذف `الموسم/الجزء` suffix) ويحدّث سيرفرات الحلقات من full JSON → 387 حلقة
2. **`_fix_servers_fetch.py`:** يفتح `see.php` لكل حلقة ويستخرج سيرفرات embed → 2,940 حلقة
3. **`_fix_remaining.py`:** يحاول مجدداً مع fallback لاستخراج iframe مباشر → 144 حلقة

### النتيجة
| الملف | قبل | بعد | النسبة |
|-------|------|-----|--------|
| `data-turkish-completed.js` | 6,114 سيرفر حقيقي | 8,536 سيرفر حقيقي | **99.8%** |
| `data-turkish-ongoing.js` | 3,813 سيرفر حقيقي | 3,811 سيرفر حقيقي | **99.4%** |
| المتبقي | 38 حلقة (روابط محذوفة/فارغة من المصدر) |

### تغيير في extract_servers
`extract_servers()` في `scrape_turkish_series.py` أضيف له fallback لاستخراج iframe مباشر من `#Playerholder` عندما لا توجد قائمة `<li>`، لأن الموقع لم يعد يعرض سيرفرات متعددة.

## الفروقات عن tuktukhd
- **المصدر:** `yam.ahwaktv.net` بدلاً من `tuktukhd.com`
- **فك التشفير:** لا يوجد تشفير لروابط المشاهدة (روابط مباشرة)
- **التصنيف:** أسوية (كورية، صينية، يابانية) + تركية
- **التوزيع:** مكتمل/مستمر في ملفين منفصلين لكل تصنيف
