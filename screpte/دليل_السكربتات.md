## 🅱 توكتوك (tuktukhd.com)

### 📁 `screpte/tuktukhd/tuktuk/` — الأفلام

| السكربت | دوره | طريقة التشغيل |
|---------|------|---------------|
| `scripte-film.py` | سحب الأفلام الأجنبية من tuktukhd | `python scripte-film.py --all -s 1 -e 5 -o "foreign.json"` |
| | | `-p 1 -sm 1 -em 10` (نطاق أفلام من صفحة) |
| | | `--list -s 1 -e 1` (عرض قائمة فقط) |
| | | `-i` (وضع تفاعلي) |

## 🧹 سير العمل النموذجي

### أهواك — تحديث المسلسلات التركية
```powershell
python "screpte/ahwak/0606206/scrape_turkish_series.py" --resume --workers 4
python "screpte/ahwak/0606206/scrape_turkish_series.py" --convert --merge
```

### أهواك — تحديث المسلسلات الأسوية
```powershell
.\update_ahwak.ps1
```

### توكتوك — تحديث الأفلام
```powershell
.\update_all.ps1
```

### توكتوك — تحديث المسلسلات الأجنبية
```powershell
.\update_series.ps1
```

### توكتوك — استخراج يدوي لأفلام + دمج
```powershell
python "screpte/tuktukhd/tuktuk/scripte-film.py" --all -s 1 -e 5 -o "screpte/tuktukhd/tuktuk/foreign.json"
python merge_tuktuk_to_data.py
```

### توكتوك — فك تشفير روابط التحميل
```powershell
python "screpte/tuktukhd/tuktuk/_decode_downloads.py"
```
