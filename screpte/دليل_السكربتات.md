## 1. StreamIMDb — أفلام أجنبية (70,000+)

**المجلد:** `screpte/streamimdb/`
**المصدر:** streamimdb.ru
**البيانات:** `screpte/streamimdb/data/streamimdb_movies.json`
**الخرج:** `data-streamimdb.js`

```powershell
# تشغيل سريع
.\scrape_all.ps1 -Site streamimdb

# أو يدوي:
python -X utf8 screpte/streamimdb/scrape_streamimdb.py --embeds-only --resume --parallel 5
python -X utf8 screpte/streamimdb/convert_to_js.py
git add -A && git commit -m "تحديث" && git push
```

### خيارات السكراب الرئيسي:

| الخيار | الوصف |
|--------|-------|
| `--listings-only` | مرحلة التصفح فقط |
| `--embeds-only` | مرحلة روابط المشاهدة فقط |
| `--resume` | استئناف من آخر حفظ |
| `--fresh` | مسح الحالة والبدء من جديد |
| `--page-from N` | بدء التصفح من صفحة N |
| `--page-to N` | التوقف عند صفحة N |
| `--from N --to N` | نطاق معالجة embed |
| `--parallel N` | عدد الخيوط (افتراضي 5) |

### أمثلة:

```powershell
# تشغيل التصفح (صفحات 1-500)
python screpte/streamimdb/scrape_streamimdb.py --listings-only --page-to 500

# تشغيل الروابط من الفيلم 5000 إلى 10000
python screpte/streamimdb/scrape_streamimdb.py --embeds-only --from 5000 --to 10000 --resume

# عرض المساعدة
python screpte/streamimdb/scrape_streamimdb.py --help
```

### retry_embeds.py — إعادة محاولة الروابط الفاشلة:

```powershell
python screpte/streamimdb/retry_embeds.py
```

## 3. Cimafre — أفلام عربية (419 فيلم، 11 سيرفر لكل فيلم)

**المجلد:** `screpte/cimafre/`
**المصدر:** cimafre.site
**البيانات:** `screpte/cimafre/data/arabic_movies.json`
**الخرج:** `data-cimafre.js`

```powershell
# تشغيل سريع
.\scrape_all.ps1 -Site cimafre

# أو يدوي:
python -X utf8 screpte/cimafre/scrape_cimafre.py --to 999
python -X utf8 screpte/cimafre/scrape_cimafre.py --servers --parallel 10
python -X utf8 screpte/cimafre/convert_to_js.py
git add -A && git commit -m "تحديث" && git push
```

### الخيارات:

| الخيار | الوصف |
|--------|-------|
| `--from N` | بدء التصفح من صفحة N |
| `--to N` | التوقف عند صفحة N |
| `--servers` | جلب السيرفرات فقط (11 سيرفر/فيلم) |
| `--parallel N` | عدد الخيوط (افتراضي 5) |

### ملاحظة مهمة:
السيرفرات تُجلب من `play.php?vid={vid}` (وليس من `watch.php` أو `category.php` المحمية بـ Cloudflare).

## 4. TukTukHD — أفلام أجنبية ومدبلجة ومسلسلات أجنبية

**المجلد:** `screpte/tuktukhd/`
**المصدر:** tuktukhd.com
**الملفات:**
- `screpte/tuktukhd/tuktuk/scripte-film.py` — أفلام أجنبية
- `screpte/tuktukhd/tuktuk/scripte-dubbed.py` — أفلام مدبلجة
- `screpte/tuktukhd/scrape_foreign_series.py` — مسلسلات أجنبية

**الخرج:** `data-foreign.js`, `data-dubbed.js`, `data-foreign-series.js`

```powershell
# تشغيل سريع (الكل)
.\scrape_all.ps1 -Site tuktuk

# أو يدوي:
python "screpte/tuktukhd/tuktuk/scripte-film.py" --all -s 1 -e 5 -o "screpte/tuktukhd/tuktuk/foreign.json"
python "screpte/tuktukhd/tuktuk/scripte-dubbed.py" --all -s 1 -e 5 -o "screpte/tuktukhd/tuktuk/dubbed.json"
python "screpte/tuktukhd/scrape_foreign_series.py" --resume --workers 4
git add -A && git commit -m "تحديث" && git push
```

### خيارات scripte-film.py و scripte-dubbed.py:

| الخيار | الوصف |
|--------|-------|
| `-s N, --start N` | صفحة البداية |
| `-e N, --end N` | صفحة النهاية |
| `-p N, --page N` | رقم صفحة محددة |
| `-sm N, --start-movie N` | بداية الفيلم في الصفحة |
| `-em N, --end-movie N` | نهاية الفيلم في الصفحة |
| `--all` | كل الأفلام في نطاق الصفحات |
| `--list` | عرض القائمة فقط |
| `-o FILE` | ملف الإخراج |
