## 1. StreamIMDb — أفلام أجنبية (70,000+)

**المجلد:** `screpte/streamimdb/`
**المصدر:** streamimdb.ru
**البيانات:** `screpte/streamimdb/data/streamimdb_movies.json`
**الخرج:** `data-streamimdb.js`

```powershell
# تشغيل سريع (pages 1-100)
.\movies\run-streamimdb.ps1 -StartPage 1 -EndPage 100

# أو يدوي:
python -X utf8 screpte/streamimdb/scrape_streamimdb.py --page-from 1 --page-to 100 --resume --parallel 5
python -X utf8 screpte/streamimdb/convert_to_js.py
git add -A && git commit -m "تحديث" && git push
```

### خيارات السكراب الرئيسي:

| الخيار | الوصف |
|--------|-------|
| `--listings-only` | مرحلة التصفح فقط |
| `--embeds-only` | مرحلة روابط المشاهدة فقط |
| `--resume` | استئناف من آخر حفظ |
| `--fresh` | مسح الحالة والبدء من جديد |
| `--page-from N` | بدء التصفح من صفحة N |
| `--page-to N` | التوقف عند صفحة N |
| `--from N --to N` | نطاق معالجة embed |
| `--parallel N` | عدد الخيوط (افتراضي 5) |

### أمثلة:

```powershell
# تشغيل التصفح (صفحات 1-500)
python screpte/streamimdb/scrape_streamimdb.py --listings-only --page-to 500

# تشغيل الروابط من الفيلم 5000 إلى 10000
python screpte/streamimdb/scrape_streamimdb.py --embeds-only --from 5000 --to 10000 --resume

# عرض المساعدة
python screpte/streamimdb/scrape_streamimdb.py --help
```

### retry_embeds.py — إعادة محاولة الروابط الفاشلة:

```powershell
python screpte/streamimdb/retry_embeds.py
```

## 5. إصلاحات وحلول

### مشكلة: المسلسلات/الأفلام الجديدة لا تظهر في "مضاف حديثاً"

**السبب:** "مضاف حديثاً" يرتب حسب `dateAdded` تنازلياً. العناصر الجديدة تُضاف بدون هذا الحقل.

**الحل:** سكريبتات الدمج تُضيف تلقائياً `dateAdded` للعناصر الجديدة. إذا أردت إصلاح ملف موجود:

```powershell
python -c "
import json, re, time
base = int(time.time())
for f in ['data-foreign.js', 'data-foreign-series.js', 'data-turkish-completed.js', 'data-turkish-ongoing.js', 'data-asian-series-completed.js', 'data-asian-series-ongoing.js']:
    with open(f, 'r', encoding='utf-8') as fh:
        c = fh.read()
    m = re.search(r'(const\s+\w+\s*=\s*)(\[.*?\])\s*;\s*$', c, re.DOTALL)
    if not m: continue
    items = json.loads(m.group(2))
    for i, item in enumerate(items):
        item['dateAdded'] = base - len(items) + i
    with open(f, 'w', encoding='utf-8') as fh:
        fh.write(m.group(1) + json.dumps(items, ensure_ascii=False) + ';')
    print(f'{f}: {len(items)} items')
"
```

### مشكلة: مسار الحفظ الخاطئ (data/ مجلد فرعي)

**السبب:** بعض سكريبتات الدمج تحفظ في `data/data-*.js` (مجلد فرعي) بينما `index.html` يقرأ من الجذر `data-*.js`.

**الحل:** أصلحنا مسار `DATA_DIR` في:
- `screpte/ahwak/0606206/merge_turkish_to_data.py`
- `screpte/ahwak/ASSIAN/merge_asian_to_data.py`
- `merge_tuktuk_to_data.py`
- `movies/run-tuktuk-films.ps1` (أُضيف `-u`)

### مشكلة: خرج السكريبت لا يظهر في PowerShell

**السبب:** Python يخبّئ الخرج عند تشغيله من PowerShell.

**الحل:** أُضيف `write_through=True` في `scripte-film.py` و `-u` في `run-tuktuk-films.ps1`.
